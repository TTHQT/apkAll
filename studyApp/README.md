## 🤔 What Does It Do?

StudyApp helps you build a personal learning workflow:

**Take notes → Make quizzes → Practice and review**

It is useful for exam preparation, flashcards, memorization, and structured review.

### 📖 Knowledge Base — Take Notes

- Create **knowledge cards** like digital flashcards
- Organize cards into **groups** such as Math, English, History, etc.
- Use **full-text search** to quickly find what you need
- Import **TXT or DOCX files** and let AI extract key points automatically
- Merge groups without moving cards manually one by one

### 📝 Exam Papers — Make Your Own Tests

- Create custom **exam papers**
- Supports **5 question types**: Single Choice, Multiple Choice, True/False, Fill in the Blank, Short Answer
- Link each question to a knowledge card for easier review
- Use AI to generate a full exam from selected knowledge cards

### 🧠 Quiz Mode — Practice and Review

- Practice with a specific paper
- Mix questions from multiple papers
- Randomly pick questions from the whole question bank
- Auto-save answers and auto-grade objective questions
- View correct answers, explanations, and linked knowledge cards after submission

### 🌐 Bonus Feature

Built-in **HTTP server**: open `http://your-phone-ip:8080` on your computer browser under the same Wi-Fi and type answers with a real keyboard.

## 📲 How to Install

Download the latest APK from the [Releases](../../releases) page and install it on your Android phone.

Minimum Android version: **Android 8.0**.

> Make sure **Install from unknown sources** is enabled before installing.

---

## 💡 AI Features

The core features work **fully offline**.

AI features such as auto-extracting knowledge points and generating exams require an API Key. The app currently supports DeepSeek, and you can configure it in the app settings.
